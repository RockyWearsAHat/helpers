## Release Scope

Makes installs **just work with no external toolchain**. `helpers build` (run by every
install path) now downloads a prebuilt `helpers-native` binary for your platform from the
release, instead of compiling Rust/C on your machine. Compilation becomes an automatic
fallback. Also folds `git-cs-grade` into that single binary.

## Highlights

- **Zero-toolchain install.** Prebuilt `helpers-native` binaries are published for
  macOS (universal), Linux x86_64 / aarch64 (glibc), Linux x86_64 (musl/Alpine), and
  Windows x86_64 / arm64. `helpers build` detects your platform and downloads the matching
  one — no Rust, no MinGW, no C compiler required. If there's no prebuilt for your platform
  or the download fails, it **falls back to building from source** (`helpers build --from-source`
  forces that).
- **One binary.** `git-cs-grade` is now folded into `helpers-native` (busybox dispatch, like
  the other `git-*` CLIs), so there's a single artifact to ship and install. `helpers grade`
  and the `git-cs-grade` symlink both route to it. The legacy Node `git-cs-grade.js` fallback
  is removed.

## Details

- helpers CLI: `fetchPrebuiltNative()` downloads/installs the per-host release tarball;
  `provisionNative()` is prebuilt-first with a source fallback; `helpers doctor`'s toolchain
  checks are now informational (only relevant for `--from-source`). The optional C launcher
  is still built locally (it bakes install-specific paths, so it can't be a portable prebuilt)
  and degrades to running Node directly when no C compiler is present.
- CI: a new `build-natives` matrix cross-builds `helpers-native` for all targets using plain
  `cargo --target` + cross-linkers (no Docker), and `publish-release` attaches the tarballs
  and their checksums.

## Validation

cs-grade merge verified (18 tools, grading output, exit codes); full
`x86_64-pc-windows-gnu` and `macos-universal` cross-builds produce working binaries; the
prebuilt-miss → source-build fallback works; cargo (25 tests) and the JS/shell suite (28/28)
stay green.
