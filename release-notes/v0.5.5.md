## Fix

- Add the `helpers` CLI symlink to `.gitignore`. It's a busybox symlink to
  `helpers-native` recreated by `helpers build`/install (like the `git-*`
  symlinks), but it was missing from the ignore list — so on a dev machine where
  the repo dir is also the install dir, the installed symlink showed as untracked
  and could be committed by accident. (The other symlinks were already ignored.)
