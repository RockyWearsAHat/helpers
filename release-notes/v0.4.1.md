## Release Scope

This release makes the standalone installer produce a complete, working install,
adds a self-update command, and fixes GitHub release publishing so release assets
are actually attached.

## Highlights

- **Working standalone installer.** `Helpers-Installer.sh` previously staged an
  incomplete tree — it never installed the `helpers` CLI or built the native
  binary (the `fetch_mcp_runtime` build path was dead code with a stale module
  list). The packaged release tarball now ships the `helpers` CLI, the
  `helpers-serverd.js` daemon, the `helpers-mcp.c` launcher source, the agent
  config bundle, and the `native/` + `cs-grade/` Rust crate sources. After
  unpacking, the installer runs `helpers build` (compiles `helpers-native`, the
  ported `git-*` CLIs, `git-cs-grade`, and the fast C launcher) and
  `helpers install` to register the MCP server with detected agents.
- **Self-update.** New `helpers update` upgrades to the latest GitHub release —
  `git pull` for a clone, or download-and-unpack the release tarball for a
  packaged install — then rebuilds the native tools and re-registers. `helpers
  update --check` reports whether an update is available without applying it, and
  `helpers status` shows a non-blocking "update available" hint, refreshed in the
  background at most once a day. Release discovery lists `/releases` and takes the
  highest version (every release here is a `0.x` prerelease, so `/releases/latest`
  does not apply).
- **Clean release publishing.** The release workflow attached zero assets because
  `upload-artifact@v4` strips the common `dist/` path prefix, so the downloaded
  files landed at `release-assets/<file>` while `action-gh-release` looked under
  `release-assets/dist/<file>`. The release file globs (and the npm publish path)
  now point at the correct location, so the tarball, installer, checksums, `.deb`,
  `.pkg`, `.tgz`, Homebrew formula, AUR metadata, and VSIX attach to the release.

## Upgrading

```
helpers update          # from an existing install
```

Or reinstall from scratch:

```
curl -fsSL https://raw.githubusercontent.com/RockyWearsAHat/github-shell-helpers/main/Helpers-Installer.sh | bash
```

Building the native tools requires a Rust toolchain (https://rustup.rs); without
it the files still install but the Rust-only MCP tools won't run.
