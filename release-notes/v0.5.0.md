## Release Scope

Repository renamed to **`RockyWearsAHat/helpers`** and the README rewritten for the
single-binary, Node-free reality. No functional change to the tools.

## Changes

- **Repo renamed** `github-shell-helpers` → `helpers`. All in-code repo URLs
  (release downloads, raw, API, community cache) now point to the new slug; GitHub
  redirects the old name, so existing links keep working.
- **README rewritten** to reflect Helpers as one prebuilt native binary: no Node, no
  toolchain, prebuilt for every major platform, install-tested on macOS/Linux/Windows
  in CI, with the `helpers` CLI + MCP server + git-* helpers all in the one binary.

Release asset and package names (`github-shell-helpers-*`, the npm/deb package names)
are unchanged to avoid breaking existing package channels.
