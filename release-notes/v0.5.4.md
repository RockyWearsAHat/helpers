## Package managers: Homebrew, Scoop, Winget, and apt

Helpers now publishes to every major package manager, each delivering the same
prebuilt native binary (no Node, no toolchain).

## Changes

- **Homebrew** — formula rewritten to install the prebuilt macOS-universal
  binary (was stale: it installed the removed Node tree). Generated at release
  time against the real binary SHA-256; published to a tap when configured.
- **Scoop** — new manifest (`scoop/helpers.json`, x64 + arm64) consuming a new
  Windows `.zip` artifact; `helpers` shim runs the CLI.
- **Winget** — new portable-in-zip manifests (`RockyWearsAHat.Helpers.*`).
- **apt** — new signed APT repository builder + a `Publish APT repo` workflow
  that pushes a (GPG-signed) `dists/`+`pool/` tree to GitHub Pages.
- Windows builds now emit a `.zip` alongside the `.tar.gz`; all package
  manifests are generated in the release-publish job (where per-platform
  checksums exist) and attached as release assets.
- Release credential plumbing gains `ENABLE_APT_PUBLISH` + `APT_GPG_PRIVATE_KEY`
  (auto-enabled when the key is present), wired through
  `scripts/setup-github-release-config.sh` and the example env.

## Enabling publication

Fill `.github/release-config.local.env` (gitignored) and run
`scripts/setup-github-release-config.sh` to push the secrets/vars to GitHub.
Final delivery to external homes still needs you to provide: a Homebrew tap
repo + token, a Scoop bucket repo, a winget-pkgs PR, and a GPG key + GitHub
Pages for apt.
