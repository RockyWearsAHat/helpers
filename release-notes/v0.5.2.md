## Release Scope

`helpers grade` (and `git-cs-grade`) now always runs the **full suite** — every
category from both courses — instead of auto-detecting a single course. There is no
course option or detection.

## Changes

- New "full" rubric scoring all seven categories every time (weights sum to 100):
  Object-oriented design & structure, Data structures & complexity, Tests & coverage,
  Documentation & Javadoc, Code style & cleanliness, Correctness & build, and
  Abstraction & encapsulation.
- `helpers grade <path> [--json]` grades the full suite; the per-course rubrics
  remain available in the library for the standalone tool/tests.
