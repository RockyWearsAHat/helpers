# Helpers v0.3.0

## Highlights

- Added git-copilot-devops-audit helper for Copilot audits
- git-upload now auto-detects repo test commands
- Added regression tests covering git-upload detection and states
- Added macOS build and test GitHub Actions workflows

## Installer Assets

- `github-shell-helpers-0.3.0.pkg` installs the commands and man pages into `/usr/local` on macOS.
- `Helpers-Installer-0.3.0.sh` is the versioned standalone installer asset for manual download or mirroring.
