# Helpers v0.2.0

## Highlights

- VS Code extension surfaces community cache controls
- Installer now auto-builds and installs helper VSIX
- build-vsix syncs VERSION before packaging the extension
- Community cache submission script enforces updated participation modes

## Installer Assets

- `github-shell-helpers-0.2.0.pkg` installs the commands and man pages into `/usr/local` on macOS.
- `Helpers-Installer-0.2.0.sh` is the versioned standalone installer asset for manual download or mirroring.
