# Helpers v0.1.5

## Highlights

- Auto-create missing release notes on AI upload.
- Pull repo guidance into AI upload prompts.
- Block version bumps missing matching release notes.
- Document versioned release-note generation behavior.

## Installer Assets

- `github-shell-helpers-0.1.5.pkg` installs the commands and man pages into `/usr/local` on macOS.
- `Helpers-Installer-0.1.5.sh` is the versioned standalone installer asset for manual download or mirroring.
