# Helpers v0.1.4

This release adds a dedicated GitHub Copilot DevOps audit helper and improves installer support for getting the audit workflow usable in VS Code faster.

## Highlights

- Added `git copilot-devops-audit` for installing and running the Copilot customization audit workflow from any cloned copy of this repo.
- The macOS installer can now refresh the VS Code audit setup for the logged-in user after install.
- The script installer can optionally help install or update the GitHub Copilot VS Code extensions and the global audit surfaces.

## Installer Assets

- `github-shell-helpers-0.1.4.pkg` installs the commands and man pages into `/usr/local` on macOS.
- `Helpers-Installer-0.1.4.sh` is the versioned standalone installer asset for manual download or mirroring.

## Notes

This release body is now sourced from `release-notes/v0.1.4.md`, so future releases can be updated without editing the main README or the GitHub Actions workflow.
