## Add

- `scripts/build-homebrew-core-formula.sh` — generates a from-source Homebrew
  formula (builds the native crate with `cargo`, `depends_on "rust"`) suitable
  for submitting to **Homebrew/homebrew-core**. This differs from the binary tap
  formula produced by `build-pkg-manifests.sh`: homebrew-core builds its own
  bottles from source, so it won't accept a prebuilt-binary formula.

  Run `bash scripts/build-homebrew-core-formula.sh` to (re)generate
  `dist/homebrew-core/helpers.rb` for the current `VERSION`.

- `search_web` gains a `search_type` parameter — `web`, `images`, or `both`
  (default `both`). Image results decode Google Images entries into the direct
  image URL plus their source page; `both` returns one labeled block per kind.

## Fix

- `search_web`: once you solve the "I'm not a robot" CAPTCHA, the visible Chrome
  window now closes itself instead of lingering. The graceful shutdown flushes the
  cleared-CAPTCHA cookie to the shared profile, so subsequent searches stay
  automated.
